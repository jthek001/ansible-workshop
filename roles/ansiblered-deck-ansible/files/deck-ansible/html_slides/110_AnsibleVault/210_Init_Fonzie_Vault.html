ANSIBLE VAULT: FONZIE (encrypted variable)

Use ansible-vault to create a new encrypted variable string.

Ansible Terminal
 ansible-vault encrypt_string 'Jumped The Shark' --name
 'fonzie_secret'

In this example, we are creating an encrypted string that will be placed into a non-encrypted YAML file.
